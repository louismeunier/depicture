<div align="center">
    <h1>depicture</h1>
    <p>
    CLI utility that parses colors in a picture.
    </p>
</div>

## Usage

```
    depicture <image/path> [flags]

    Flags:
    -h, --help             help for depicture
    -c, --max-colors int   maximum colors to return (default 3)
    -r, --remote           whether the file is remote
```
